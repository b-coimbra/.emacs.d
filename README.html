1 Installation

Clone this repo into your .emacs.d: 

$ git clone https://github.com/0-l/megumacs.git ~/.emacs.d --recurse-submodules

1.1 Font

The font currently set in the config is Iosevka.

1.2 Icons

If you care about fancy icons, install the fonts provided by all the icons: 

M-x all-the-icons-install-fonts

1.3 Syntax checking/linting

Most IDE-like features are provided by the LSP package. For it to work properly, you need to install the appropriate language servers.
